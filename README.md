A backtesting framework for evaluating sports betting strategies
using historical data, with an emphasis on risk-adjusted performance
and reproducible analysis.

### Overview
This project provides tooling to simulate betting strategies over
historical datasets and evaluate their performance across multiple
metrics rather than raw win rate alone.

### Features
- Strategy backtesting on historical sports data
- Performance metrics including ROI, win rate, and max drawdown
- Bankroll trajectory visualization
